Instalación de Astro

Con la terminal abierta en la carpeta de tus proyectos, ejecuta el asistente oficial de Astro:

npm create astro@latest

Pasos del Asistente

  • Directory: Elige el nombre de la carpeta de tu proyecto.
  • Template: Selecciona Empty para un inicio limpio.
  • Dependencies: Dile que sí (Install dependencies).
  • Git: Inicializa el repositorio para control de versiones.
