Conceptos Clave de Astro

1. Pages (Páginas)

Cualquier archivo .astro dentro de src/pages/ se convierte automáticamente en una ruta URL accesible en la web.

2. Layouts (Plantillas)

Son componentes especiales destinados a envolver tus páginas para proveer la estructura básica del documento (<html>, <head>, <body>) mediante la etiqueta <slot />.

3. Componentes

Bloques de código HTML/CSS reutilizables (como botones, menús o tarjetas) que creas en src/components/ para evitar duplicar código.
