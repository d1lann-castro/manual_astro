Despliegue en GitHub Pages

Para desplegar tu sitio Astro en GitHub Pages, sigue estos pasos:

    Configura tu repositorio de GitHub para alojar el sitio.>
  1. Asegurate de que tu archivo astro.config.mjs tenga la propiedad base.
  2. Agrega un script de despliegue en tu archivo package.json como: start y deploy.
  3. Ejecuta el comando de despliegue npm run deploy

Después de ejecutar el comando, tu sitio estará disponible en la URL configurada en Github Pages.
