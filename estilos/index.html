Estilos con Tailwind CSS

Tailwind CSS funciona mediante clases de utilidad directamente aplicadas en las etiquetas HTML. En este proyecto aplicamos los siguientes conceptos:

Layout (Contenedores)

max-w-3xl mx-auto flex block grid

Espaciados

p-4 (padding) my-6 (margin vertical)

Tipografía

text-3xl font-extrabold tracking-wide

Visuales

shadow-sm rounded-md bg-white
